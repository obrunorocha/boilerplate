Projeto em Next.js
https://nextjs.org/docs/getting-started

Ferramentas usadas
Yarn
https://yarnpkg.com/

Node
https://nodejs.org/en/

Comando de criação de Projeto
npx create-next-app / yarn create next-app

Comando para rodar o Projeto
npx run dev / yarn dev

----------------------------------------
Typescript
https://nextjs.org/docs/basic-features/typescript

Configurando o Typescript com o Next
Comando: touch tsconfig.json e rodar: npx run dev / yarn dev

Instalar as dependencias para o funcionamento do Typescript
yarn add --dev typescript @types/react

Após rodar o comando: npx run dev / yarn dev 

----------------------------------------
EditorConfig
https://editorconfig.org/

Define certas regras para o editor, onde toda vez que crie ou inicie um novo arquivo ele siga essas regras.

----------------------------------------

https://eslint.org/
